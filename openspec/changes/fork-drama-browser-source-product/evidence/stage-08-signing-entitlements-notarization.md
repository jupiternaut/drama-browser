## Public Release Boundary

This stage does not claim public notarization. Public distribution still requires:

- final non-local bundle id;
- Apple Developer ID Application signing identity;
- hardened runtime signing for app, helpers, frameworks, executables, and dylibs;
- notarization submission and accepted ticket;
- stapling and Gatekeeper `spctl` acceptance;
- update metadata signed by Drama-owned keys.

Until those are done, this artifact is internal-only and updater remains disabled.
